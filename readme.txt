Earn money by adding widget ads to your website or buy avertising space without never leaving your website.

== Description ==

This plugin is used to show ads on your website and get paid. It use an external API from TouTrix but you never have to leave your WordPress website. You can create your TouTrix account easily in your admin panel.

Add Ads Widget to your page to show ads and earn money, check your stats and everything without never leaving your website.

For Advertiser, you manage all your ads in your WordPress admin panel. Stats are available and much more.


This plugin use this TouTrix API, but did we said you never have to leave your website?

http://serv.toutrix.com/explorer/


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp_toutrix` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add widget or create your ads
